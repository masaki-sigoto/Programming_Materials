# WordPress自動デプロイ機能のテスト

## これは何？

この記事は、**GitHub Actions** を使った WordPress 自動デプロイ機能のテストです。

ローカルで Markdown ファイルを編集し、Git にプッシュするだけで、自動的に WordPress に投稿されることを確認します。

## テスト内容

### ✅ 確認項目

1. **Markdown から HTML への変換**
2. **GitHub Actions の自動実行**
3. **WordPress REST API の接続**
4. **投稿の自動作成**

### 🎯 期待される結果

この記事が WordPress の**下書き一覧**に表示されれば、自動デプロイは成功です！

## 技術スタック

- **GitHub Actions**: ワークフロー自動実行
- **Node.js**: デプロイスクリプト
- **WordPress REST API**: コンテンツ投稿

## まとめ

このテストが成功すれば、今後は：

- ローカルで記事を書く
- Git にプッシュ
- **自動的に WordPress に投稿される！**

という効率的なワークフローが実現します。
