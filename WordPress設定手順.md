# WordPress固定ページ設定手順（iframe埋め込み）

## 📋 準備

- ✅ GitHubにプッシュ完了
- ✅ GitHub Pages公開済み（https://masaki-sigoto.github.io/Programming_Materials/）
- ✅ 埋め込みコード準備完了（`wordpress_embed_code.html`）

---

## 🚀 設定手順

### ステップ1: WordPressにログイン

1. WordPressの管理画面にアクセス
   - `https://your-site.com/wp-admin/`
2. ユーザー名とパスワードを入力してログイン

---

### ステップ2: 新しい固定ページを作成

1. 左側メニューから **「固定ページ」** → **「新規追加」** をクリック

2. **ページタイトル**を入力
   ```
   プログラミング学習ガイド
   ```
   または
   ```
   小学生のプログラミング入門
   ```

3. **パーマリンク（URL）**を設定（推奨）
   - タイトル下の「URLスラッグ」を編集
   - 例: `programming-guide` または `programming-learning`

---

### ステップ3: カスタムHTMLブロックを追加

1. **ブロックを追加**
   - ページエディタの左上の「+」ボタンをクリック

2. **「カスタムHTML」**を検索して選択
   - 検索窓に「カスタム」または「HTML」と入力
   - 「カスタムHTML」ブロックが表示されるのでクリック

---

### ステップ4: 埋め込みコードを貼り付け

1. **コードファイルを開く**
   - デスクトップの `wordpress_embed_code.html` をテキストエディタで開く

2. **コード全体をコピー**
   - `Cmd + A`（全選択）→ `Cmd + C`（コピー）

3. **WordPressのカスタムHTMLブロックに貼り付け**
   - ブロック内をクリック
   - `Cmd + V`（貼り付け）

4. **プレビューで確認**（オプション）
   - ブロック右上の「プレビュー」切り替えボタンをクリック
   - 埋め込みページが表示されることを確認

---

### ステップ5: ページ設定（オプション）

#### 全幅レイアウトにする（推奨）

1. 右側パネルの **「ページ」**タブを選択

2. **「テンプレート」**セクションを探す

3. 以下のいずれかを選択（テーマによって異なる）
   - 「全幅」
   - 「Full Width」
   - 「幅広」
   - 「ランディングページ」

#### サイドバーを非表示にする

- テーマによっては「サイドバー」の設定があります
- **「サイドバーなし」**を選択

---

### ステップ6: 公開

1. **下書き保存**
   - 右上の「下書き保存」をクリック（安全のため）

2. **プレビュー確認**
   - 「プレビュー」ボタンをクリック
   - 新しいタブで表示を確認

3. **公開**
   - 問題なければ右上の **「公開」** ボタンをクリック
   - 確認ダイアログが出たら再度 **「公開」** をクリック

---

### ステップ7: メニューに追加（オプション）

1. **外観 → メニュー**
   - 左側メニューから「外観」→「メニュー」

2. **固定ページを追加**
   - 左側の「固定ページ」セクションで作成したページを探す
   - チェックボックスにチェック
   - 「メニューに追加」をクリック

3. **位置を調整**
   - ドラッグ&ドロップでメニュー内の位置を調整

4. **メニューを保存**
   - 右下の「メニューを保存」をクリック

## 🎨 カスタマイズ（必要な場合）

### iframeの高さを調整

コード内の以下の部分を編集：
```css
.programming-guide-iframe {
    width: 100%;
    min-height: 3000px; /* ← この値を変更 */
    border: none;
}
```

### モバイル時の高さを調整

```css
@media (max-width: 768px) {
    .programming-guide-iframe {
        min-height: 4000px; /* ← この値を変更 */
    }
}
```

### 背景色を変更

```css
.programming-guide-container {
    background: #ffffff; /* ← この色を変更 */
}
```

## 🔧 トラブルシューティング

### iframe が表示されない

**原因**: WordPressのセキュリティ設定
**解決策**:
1. プラグイン「Enable iframe」をインストール
2. または管理者にiframeの許可を依頼

### 高さが合わない

**原因**: コンテンツの実際の高さと設定値の不一致
**解決策**:
1. `min-height` の値を増やす（例: 3000px → 4000px）
2. ブラウザの開発者ツールで実際の高さを確認

### スクロールが二重になる

**原因**: WordPressテーマとiframeのスクロールバーが重複
**解決策**:
```css
.programming-guide-iframe {
    overflow: hidden; /* 追加 */
}
```

### 更新が反映されない

**原因**: ブラウザキャッシュ
**解決策**:
1. `Cmd + Shift + R`（Mac）または `Ctrl + Shift + R`（Windows）でリロード
2. WordPressのキャッシュプラグインをクリア

## 🔄 今後の更新フロー

サイトを更新する場合：

```
1. ローカルで index.html を編集
2. git add . && git commit -m "更新内容"
3. git push second main
4. 数秒〜1分で GitHub Pages に反映
5. WordPress の iframe も自動で最新版を表示
```

**WordPress側での作業は不要**です！
